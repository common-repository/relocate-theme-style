== Description ==
Filters WordPress output to allow you to relocate your theme stylesheets.

The plugin adds an option to the "General" options page, right underneath the "Blog address" field. The plugin is designed to allow you to replicate the folder structure of the default WordPress themes directory, so that you don't have to change the option every time you change your theme.


== Installation ==

1. Upload the `relocate-theme-style` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Requirements ==

* PHP5
